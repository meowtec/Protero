# protero
Core of [MeoProxy](https://github.com/meowtec/meoproxy).

## Development
Install OpenSSL firstly.

Check whether OpenSSL has been installed.
```sh
openssl
```

Install Nodejs (>=4.2)
```
node -v
```

Install dependences.
```
npm install
```

Run test cases.
```
npm test
```
